# Debug Helper

**Biến / Variables:** `{{ERROR_MESSAGE}}`, `{{CODE}}`, `{{CONTEXT}}`

---

Tôi đang gặp lỗi sau:

```
{{ERROR_MESSAGE}}
```

Code liên quan:
```
{{CODE}}
```

Context thêm: {{CONTEXT}}

Hãy giúp tôi:
1. Giải thích nguyên nhân lỗi
2. Đưa ra cách sửa cụ thể (kèm code)
3. Giải thích cách tránh lỗi tương tự trong tương lai
